What is DexTrust

DexTrust is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making wallets a safer choice compared to desktop or browser extension wallets.

What is Wallet Connect?

WalletConnect is a protocol that lets you interact with Decentralized Applications (DApps), opening up exciting new use cases for your crypto. For example, with WalletConnect, you can:

  • Buy and sell NFTs on marketplaces like OpenSea
  • Trade crypto peer-to-peer using Decentralized Exchanges (DEXes) like Uniswap and SushiSwap
  • Lend, borrow, and get yields from your digital assets using Decentralized Finance (DeFi) protocols like Aave and Compound
  • Buy virtual land in the Metaverse using protocols like Decentraland
