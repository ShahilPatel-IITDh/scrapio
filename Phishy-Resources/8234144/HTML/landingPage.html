Nordea ID app

Nordea ID is a mobile app that you use to log in to Nordea services and confirm actions. You can also identify yourself towards other service providers.

Read more about Nordea ID
www.nordea.fi

Login identifier

The identifier helps you confirm the right login. Check that the identifier shown by the Nordea ID app matches the one on this page.

If the identifiers do not match, ignore the request – do not approve it.

User ID

The user ID issued to you by Nordea. It contains 5-13 characters, numbers only.

Nordea ID app QR code

A QR code is a one-time code that allows you to log in without the need to enter your user ID. The QR code is used together with the Nordea ID app.

  1. Click OK to generate the code.
  2. Open the Nordea ID app on your phone or tablet.
  3. Tap “Scan QR code” in the app. This will switch on the camera for scanning the QR code.
  4. Focus the camera on the QR code. Make sure the entire code is visible within the camera view in the app.
  5. Confirm your login with your PIN or biometric identification (face or fingerprint ID).

Nordea ID app offline mode

Nordea ID is a mobile app that you use to log in to Nordea services and confirm actions. You can also identify yourself towards other service providers. Nordea ID can be used in online and offline mode. If your device is not connected to a network, you can continue to use Nordea ID by selecting Go offline.

User ID

Response code

Tap LOGIN in Nordea ID in offline mode and follow the instructions to get the response code for login.
