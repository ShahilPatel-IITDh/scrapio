Enigmail for Epyrus

The latest version of Enigmail works with Epyrus version 2.x

Download Enigmail for Epyrus

Version 3.1.4


Download GPG Signature (to verify the signature, please follow our instructions )

Changelog

To install Enigmail in Epyrus, use right click "Save Link as ..." to save the extension locally. Then navigate to the Epyrus menu Tools > Add-ons Manager and click on the small Tools button in the upper right corner and then on Install Add-on From File... .
