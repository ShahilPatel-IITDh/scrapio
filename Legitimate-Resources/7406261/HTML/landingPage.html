HotSheet Voice Search

Click the blue microphone icon, or click a mic under the Hotlist top sites list, and begin speaking when the blue icon shows red. Speak in a natural voice and say a search term as you would usually type it in.

If your web browser tab does not show a red indicator while you're talking, the recording has stopped or there is an issue preventing voice recognition. These voice features work with most web browsers (not Firefox). You may need to authorize hotsheet.com for microphone use in browser security settings.
Voice Commands
open "domain" - Open the domain.com website if it exists.
open "domain dot ext" - Say domain, the word dot, then net, tv, etc extension.

next or back - Changes the news network for the displayed headlines.

note: Voice recognition and commands may show unintended results and some editing should be expected. Edge web browsers add most punctuation automatically.
