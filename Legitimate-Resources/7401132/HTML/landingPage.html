Dryatofa , 10/01/2022

Great App but Slight Accessibility Challenge

I have found this app to be easy to use, and it appears to be accessible with screen readers. I have only come across a slight challenge that may make this app difficult to use for blind users. When you use the app to sign up for online banking, you are only given a five or ten minute session before your progress is lost. For a blind user, this is not enough time for card activation because we can't read the card number, meaning that we need to use a special software to scan the card and read the number aloud so we can type it. This process alone took me long enough to time out my session so when I finished typing the number my progress was lost. People who are blind simply can't read and type as fast as sighted people, so these sessions are not long enough. I understand that this is for security, but to make this app more accessible, I recommend thirty minute sessions. If this is not possible, another option would be to add an option to the session time out screen saying user is still there. Instead of just telling you the session will time out in one minute, it could ask if you are still there and if the user does not answer within a certain time it will automatically log off. I thank anyone who has read and considered this feedback, as well as any developers who could work to implement it.
