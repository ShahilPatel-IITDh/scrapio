Где качаться в ВоВ: Катаклизм?

Уровень Местность
1 - 5 Кезан ( Водоворот )
1 - 10 Лес Вечной Песни, Тирисфальские Леса, Дун Морог, Элвинский лес ( Восточные Королевства ); Тельдрассил, Остров Лазурной Дымки, Дуротар, Мулгор ( Калимдор )
5 - 12 Затерянные Острова ( Водоворот )
10 - 15 Западный Край ( Восточные Королевства )
10 - 20 Призрачные Земли, Серебряный Бор, Лок Модан ( Восточные Королевства ); Остров Кровавой Дымки, Темные Берега, Азшара, Северные Степи ( Калимдор );
15 - 20 Красногорье ( Восточные Королевства )
20 - 25 Предгорья Хилсбрада, Болотина, Сумеречный лес ( Восточные Королевства ); Ясеневый Лес ( Калимдор )
25 - 30 Нагорье Арати ( Восточные Королевства ); Когтистые горы ( Калимдор )
25 - 35 Тернистая долина ( Восточные Королевства )

30 - 35: Внутренние Земли ( Восточные Королевства );  П устоши, Южные Степи ( Калимдор )

44 - 48: Бесплодные Земли ( Восточные Королевства )

45 - 50: Оскверненный Лес, Танарис (Калимдор)

47 - 51: Тлеющее Ущелье ( Восточные Королевства )

49 - 52: Пылающие Степи ( Восточные Королевства )

50 - 55: Зимние Ключи, Кратер Ун`Горо,  ( Калимдор )

55 - 60: Лунная поляна, Силитус ( Калимдор )

60 - 64: Загрантопь ( Запределье )

67 - 70: Пустоверть, Долина Призрачной Луны ( Запределье )

68 - 72: Борейская Тундра, Ревущий Фьорд ( Нордскол )

71 - 75: Драконий Погост ( Нордскол )
