Blat - -port

-portSMTP is the same as -port,
use -portNNTP for NNTP (or news),
use -portPOP3 for POP3.

This parameter is used to specify a non-default TCP/IP port to connect to the server with.

Defaults; SMTP=25, NNTP=119, POP3=110

Can be stored in a profile (in the Registry) with "blat -install ".
