Apple iTunes Updated to 10.2.1 with iOS 4.3 compatibility

Apple released a new version of its mobile operating system iOS 4.3 on Wednesday. It is available for the iPhone, iPod touch, and iPad. The update is already available via iTunes. This update contains many new features and improvements including a faster Safari performance, new AirPlay features and more.

Apple has updated its PC and Mac versions of the iTunes desktop all-in-one application for "i" Devices to version 10.2.1 and the major feature is iOS 4.3 compatibility.

iTunes 10.2.1 also includes improved Home Sharing, and lets users play music from their iTunes library on any iPhone iPad or iPod touch over Wi-Fi using iOS 4.3. Any other changes to iTunes from version 10.2 are likely minor, as Apple did not detail them.
